=================
pw_spi_zephyr
=================
.. pigweed-module::
   :name: pw_spi_zephyr

``pw_spi_zephyr`` implements the :ref:`module-pw_spi` interfaces using the
`Zephyr API <https://github.com/zephyrproject-rtos/zephyr>`_.

Setup
=====
Use of this module requires setting up the Zephyr build environment for use with
Pigweed. Follow steps in :ref:`docs-quickstart-zephyr` to get setup.
